## 辅助工具
使用工具前，请明确游戏如何带来乐趣。根据作者独断而带有偏见的思考，举例如下

* 极为重复且占用精力的动作大概率没有乐趣，如不停地重复刷某一个关卡
* 研究关卡的（极限）过法大概率有乐趣
* 欣赏立绘大概率有乐趣
* 养成角色大概率有乐趣
* 看方舟沙雕图大概率有乐趣
* 看同人漫画大概率有乐趣
* 写极为复杂的自动化操作非常没有乐趣
* 作者技术太辣鸡，也没有时间

因此辅助工具目前仅限于自动化自动代理某一单一关卡。

分辨率未全适配，使用前请调整分辨率为1920x1080，并只开启一个设备的调试功能。
